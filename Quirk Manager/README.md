# Quirk Manager Update

**Objective:** Add personality-driven quirks that subtly modify the AI’s responses, making conversations feel more expressive, informal, and human-like.

---

## New Features (v2.5.6)

### Personality Quirks
- Injects small, randomized quirks into responses
- Quirks adapt to the AI’s current mood (amused, annoyed, neutral)
- Enhances personality without overwhelming core responses
- Keeps behavior unpredictable but controlled

### Slang & Expression Injection
- Replaces select words with casual slang
- Adds emojis sparingly for expressive effect
- Ensures stylistic changes remain lightweight and readable
